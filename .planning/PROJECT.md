# DnD Map Builder

## Project Context
DnD Map Builder is a desktop application built in Java using JavaFX. It allows Dungeon Masters to automatically generate and customize maps for their campaigns, saving time while retaining creative freedom.

## Core Purpose
- Automatically generate parameterized random maps.
- Allow editing of map elements (Points of Interest, text labels, borders).
- Export maps for use in D&D campaigns.

## Frameworks & Tech Stack
- **Language:** Java
- **UI Framework:** JavaFX
- **Target Platforms:** Windows, macOS, Linux (Cross-platform)

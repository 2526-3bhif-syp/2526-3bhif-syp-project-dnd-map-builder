<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| POI id uniqueness | Sequential int ids assigned during MapGrid.addPointOfInterest(); no concurrent access in Phase 3 (single-threaded JavaFX app) |
| customColor ARGB | User-provided int; no validation needed in data model (validation in UI layer when color picker saves) |
| POI x/y coordinates | User can place POIs anywhere (0 to gridWidth-1, 0 to gridHeight-1); validation in generation rules and UI editor, not in model |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | PointOfInterest immutability | mitigate | id, x, y, type, createdAt are immutable after construction; only name/description/customColor/customIcon can be edited |
| T-03-02 | Information Disclosure | POI data in memory | accept | Application is offline, local storage only; no network transmission |
| T-03-03 | Elevation of Privilege | List modification | accept | Single-threaded JavaFX; no multi-threaded access; MapGrid is only holder of truth |
</threat_model>
